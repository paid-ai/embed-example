Paid embed SDK — test harness

Mirrors Settings > Share auth in your Paid dashboard. Complete each step on the left, see the result on the right.

0 Connection

Enter your Paid API base URL, app base URL, API key, and signing secret. The API key and secret come from your Paid dashboard.

1 URLs

Copy these into your Paid app under Settings > Share auth > Configuration > Edit.

2 JWT

Mint a signed token. Optionally bind it to a specific value receipt. Leave the VR blank to mint a customer-scoped token that works for all value receipts belonging to that customer.

3 Render

Pick a value receipt and render it using the JWT from step 1. Try a different VR or customer to see the middleware reject it.

Publish value receipt
